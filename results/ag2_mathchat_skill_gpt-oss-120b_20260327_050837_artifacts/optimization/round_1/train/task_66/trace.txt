(the *binary weight* of \(n\)).  
Define  

\[

O(N)=\#\{\,1\le n\le N\mid \operatorname{wt}(n)\text{ is odd}\,\}.
\]

We are asked for  

\[
\Delta =E\bigl(2^{2017}\bigr)-O\bigl(2^{2017}\bigr).

### 1.  Reformulating the problem

Introduce the signed count  

\[
S(N)=\sum_{n=0}^{N}(-1)^{\operatorname{wt}(n)} .
\]

For each integer \(n\)

* if \(\operatorname{wt}(n)\) is even, the term contributes \(+1\);
* if \(\operatorname{wt}(n)\) is odd, the term contributes \(-1\).

Hence  

\[
S(N)=E(N)-O(N) \quad\text{(including }n=0\text{)} .
\]

Because \(0\) has weight \(0\) (even), it contributes \(+1\).  
Therefore for the *positive* integers only we have  

\[
\Delta = S\bigl(2^{2017}\bigr)-1 .
\]

So the problem reduces to evaluating \(S\bigl(2^{2017}\bigr)\).

--------------------------------------------------------------------

### 2.  Evaluating \(S(2^{k})\) for a general exponent \(k\)

Write \(k\) instead of the specific 2017; the argument works for any non‑negative integer \(k\).

All numbers from \(0\) to \(2^{k}-1\) are exactly the binary strings of length at most \(k\).  
For each fixed length \(k\) there are \(\binom{k}{j}\) strings with exactly \(j\) ones. Hence

\[
\sum_{n=0}^{2^{k}-1}(-1)^{\operatorname{wt}(n)}
   =\sum_{j=0}^{k}\binom{k}{j}(-1)^{j}
   =(1-1)^{k}=0 .
\]

Thus the signed sum over the interval \([0,2^{k}-1]\) is zero.

The interval \([0,2^{k}]\) contains one extra number, namely \(2^{k}\) itself.
Its binary representation is a single 1 followed by \(k\) zeros, so \(\operatorname{wt}(2^{k})=1\) (odd).  
Consequently its contribution is \((-1)^{1}=-1\).

Therefore  

\[
S(2^{k}) = 0 + (-1) = -1 .
\]

\[
\text{string } \longrightarrow \text{string with the first bit flipped}
\]

\[

\]

\[

so its weight is **odd**.

\[
E_1 = E_0 = 2^{2016},\qquad
O_1 = O_0+1 = 2^{2016}+1,
\]
\[
D_1 = E_1-O_1 = -1.
