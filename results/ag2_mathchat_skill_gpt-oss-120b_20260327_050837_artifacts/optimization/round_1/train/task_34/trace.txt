Output: **Solution – Choosing the correct definition of a 99 % confidence interval**

### Step‑by‑step reasoning

1. **Recall the formal definition**  
   A *confidence interval* (CI) for a population parameter (e.g., a mean, proportion, difference) is a random interval computed from a sample.  
   The *confidence level* (e.g., 99 %) is the proportion of times that this random interval will capture the **true, fixed but unknown** parameter value if we were to repeat the sampling process an infinite number of times.

2. **Interpret each option**  

   - **(A)** “99 % of the time in repeated samples, the interval would contain the true value of the parameter.”  
     This matches the definition above: the interval is random, the parameter is fixed, and the coverage probability is 0.99.

   - **(B)** “99 % of the time in repeated samples, the interval would contain the estimated value of the parameter.”  
     The *estimated* value (e.g., the sample mean) is itself random and will always lie inside the interval that was constructed from that same sample, so this statement is trivial and does not describe confidence.

   - **(C)** “99 % of the time in repeated samples, the null hypothesis will be rejected.”  
     This describes a *type‑I error* rate or power, not a confidence interval.

   - **(D)** “99 % of the time in repeated samples, the null hypothesis will not be rejected when it was false.”  
     This describes *type‑II error* (β) or power, again unrelated to confidence intervals.

3. **Select the correct option**  
   Only option **(A)** correctly captures the meaning of a 99 % confidence interval.

---

### Answer

**(A) 99 % of the time in repeated samples, the interval would contain the true value of the parameter.**
