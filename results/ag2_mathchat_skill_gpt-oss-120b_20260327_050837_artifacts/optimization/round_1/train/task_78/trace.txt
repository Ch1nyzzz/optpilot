\[
p\mid f(m+n)\iff p\mid f(m)+f(n)\qquad\text{for every }m,n\in\mathbb N

## 1.  What the condition really says  

For a positive integer \(x\) denote  

\[
\operatorname{rad}(x)=\prod_{p\mid x}p
\]

(the product of the distinct prime divisors of \(x\)).  
The given condition is equivalent to  

\[
\operatorname{rad}\bigl(f(m+n)\bigr)=\operatorname{rad}\bigl(f(m)+f(n)\bigr)
\qquad\forall m,n\in\mathbb N .\tag{1}
\]

Thus the two numbers have exactly the same set of prime divisors; they may differ
by powers of those primes.

--------------------------------------------------------------------

## 2.  Periodicity of the prime‑divisibility sets  

For a prime \(p\) put  

\[
A_{p}:=\{\,k\in\mathbb N\mid p\mid f(k)\,\}.
\]

From (1) with \(n\) replaced by \(p\) we obtain  

\[
p\mid f(m+p)\iff p\mid f(m)+f(p)\iff p\mid f(m) .
\]

Hence  

\[
k\in A_{p}\Longleftrightarrow k+p\in A_{p}\qquad(\forall k) .\tag{2}
\]

So each set \(A_{p}\) is **periodic with period \(p\)**.  
Because \(f\) is surjective there is a number \(t_{p}\) with \(f(t_{p})=p\); then
\(t_{p}\in A_{p}\). By (2) we obtain

\[
A_{p}= \{\,t_{p}+kp\mid k\in\mathbb N_{0}\,\}=t_{p}+p\mathbb N_{0}. \tag{3}
\]

Thus for every prime \(p\) there is a *single* residue class modulo \(p\)
(the class of \(t_{p}\)) that consists exactly of the arguments on which
\(p\) divides the value of \(f\).

--------------------------------------------------------------------

## 3.  The residue classes are compatible  

Take two different primes \(p\) and \(q\).  
Because \(f\) is surjective we can choose \(u\) with \(f(u)=pq\).  
Then \(u\in A_{p}\cap A_{q}\); using (3),

\[
u\equiv t_{p}\pmod p ,\qquad u\equiv t_{q}\pmod q . \tag{4}
\]
