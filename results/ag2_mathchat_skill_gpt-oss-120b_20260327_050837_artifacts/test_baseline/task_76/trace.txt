### 1.  Notation  

For a function \(f:\mathbb N\to\mathbb N\) put  

\[
a_{n}=2f(n)-n^{2}\qquad (n\ge 1).
\]

The condition of the problem is  

\[
\tag{1}
f(m)+f(n)-mn\neq 0\quad\text{and}\quad 
f(m)+f(n)-mn\mid mf(m)+nf(n)
\qquad\forall m,n\in\mathbb N .
\]

--------------------------------------------------------------------

### 2.  What we obtain from the diagonal case  

Take \(m=n\) in (1).  
Then  

\[
2f(n)-n^{2}=a_{n}\neq 0,\qquad 
a_{n}\mid 2nf(n).
\]

Because  

\[
2nf(n)-na_{n}=2nf(n)-n\bigl(2f(n)-n^{2}\bigr)=n^{3},
\]

we also have  

\[
\boxed{a_{n}\mid n^{3}}\qquad\forall n\in\mathbb N .\tag{2}
\]

Since \(f(n)\) is an integer, \(a_{n}\equiv n^{2}\pmod 2\); i.e.  

\[
a_{n}\text{ is odd when }n\text{ is odd, and even when }n\text{ is even.}\tag{3}
\]

--------------------------------------------------------------------

### 3.  The value at \(1\)

From (2) with \(n=1\) we get \(a_{1}\mid 1\); together with (3) this forces  

\[
a_{1}=1\qquad\Longrightarrow\qquad f(1)=\frac{1^{2}+a_{1}}{2}=1 .
\tag{4}
\]

--------------------------------------------------------------------

### 4.  A useful reformulation  

Write \(f(k)=k^{2}+g(k)\) and set  

\[
g(k)=\frac{a_{k}}{2}\qquad(k\ge 1).
\]

Then \(g(k)\in\mathbb Z\) and (2) becomes  

\[
\boxed{2g(k)\mid k^{3}}\qquad\forall k.\tag{5}
\]

With this notation the divisor in (1) is  

\[
\begin{aligned}
D_{m,n}&:=f(m)+f(n)-mn\\
       &=m^{2}+g(m)+n^{2}+g(n)-mn\\
       &=(m-n)^{2}+mn+g(m)+g(n).      \tag{6}
\end{aligned}
\]

Multiplying the divisibility in (1) by \(2\) and using (6) we obtain  

\[
\boxed{D_{m,n}\mid m^{3}+n^{3}+mg(m)+ng(n)}\qquad\forall m,n.\tag{7}
\]

from itertools import product

def good_function(vals):
    # vals is a tuple (f(1),…,f(N))
    N = len(vals)
    f = lambda k: vals[k-1]
    for m in range(1, N+1):
        for n in range(1, N+1):
