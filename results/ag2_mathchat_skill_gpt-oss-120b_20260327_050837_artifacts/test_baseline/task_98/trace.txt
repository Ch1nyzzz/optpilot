Output: **Solution**

For a fixed integer \(n\ge 2\) let  

\[
A_{n}= \{\,2^{n}-2^{k}\mid 0\le k<n \}.
\]

Each element of \(A_{n}\) has exactly \(n\) binary digits; all of them are
\(1\) except a single digit \(0\) (the digit in the \(k\)-th position,
counted from the right).  
The smallest element of the set is  

\[
a:=2^{n}-2^{n-1}=2^{\,n-1},
\]

and the largest one is  

\[
b:=2^{n}-1 .
\]

Because \(b\) is odd, \(\gcd(A_{n})=1\); consequently every sufficiently
large integer can be expressed as a (non‑negative) sum of elements of
\(A_{n}\).  
The problem asks for the largest integer that **cannot** be expressed in
that way – the Frobenius number of the set \(A_{n}\).

--------------------------------------------------------------------
### 1.  A lower bound

For \(n\ge 3\) consider  

\[
M_{n}=2^{\,2n-3}+1 .
\]

In binary  

\[
M_{n}=1\underbrace{0\ldots0}_{n-2}\,1\underbrace{0\ldots0}_{n-2},
\]

i.e. a block of \(n-2\) zeros separates two isolated ones.  
Every element of \(A_{n}\) consists of a block of \(n\) consecutive
ones with a single zero inside that block. Adding any number of such
blocks can never create a binary word that contains **two** isolated
zeros separated by at least one \(1\).  
Since the binary expansion of \(M_{n}\) has exactly that pattern,
\(M_{n}\) cannot be written as a sum of elements of \(A_{n}\).

Hence  

\[
g(n)\ge M_{n}=2^{\,2n-3}+1\qquad (n\ge 3).
\]

For the exceptional case \(n=2\) we have \(A_{2}=\{2,3\}\); the
well‑known Frobenius number for \(\{2,3\}\) is \(1\).

# Verify the claimed Frobenius number for the sets A_n
# ---------------------------------------------------
# For each n we generate the set A_n = {2**n - 2**k : 0 <= k < n}
# and then compute all sums of up to a reasonable number of terms.
# Because the Frobenius number grows like 2^{2n-3}, we only need to
# check integers up to that value plus a safety margin.

def frobenius_candidate(n):
    """return the claimed largest non‑representable integer for given n."""
    if n == 2:
        return 1
    return 2**(2*n - 3) + 1

def generate_A(n):
    """return the list of elements of A_n."""
    return [2**n - 2**k for k in range(n)]

def largest_nonrepresentable(n, limit_factor=2):
    """
    Compute the actual largest integer that cannot be written as a sum
    of elements of A_n (repetitions allowed).  We search up to
    limit_factor * claimed value, which is more than enough for the
    small n we test.
    """
    A = generate_A(n)
    # dynamic programming: reachable[i] = True if i can be formed
    max_check = limit_factor * frobenius_candidate(n)
    reachable = [False] * (max_check + 1)
    reachable[0] = True                     # empty sum (we will ignore 0 later)

    for i in range(1, max_check + 1):
        # try subtracting each a in A; if any remainder is reachable, i is reachable
        reachable[i] = any(i - a >= 0 and reachable[i - a] for a in A)

    # find the largest i > 0 that is NOT reachable
    for i in range(max_check, 0, -1):
        if not reachable[i]:
            return i
    return None   # everything reachable (should not happen)

# Test for several values of n
for n in range(2, 7):
    cand = frobenius_candidate(n)
    actual = largest_nonrepresentable(n)
    print(f"n = {n:2d} | claimed = {cand:,} | actual = {actual:,} | match = {cand == actual}")
