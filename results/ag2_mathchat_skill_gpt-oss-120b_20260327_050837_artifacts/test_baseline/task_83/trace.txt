### 1.  What the game does to the multiset of numbers  

* Let  

\[
S=\sum_{i}a_i\qquad\text{and}\qquad N=\#\{a_i\}
\]

be respectively the total sum and the number of entries on the board.  
Initially  

\[
S_0=2020,\qquad N_0=2020 .
\]

* In one round the two chosen numbers \(x,y\) are erased and B writes either  

\[
x+y\qquad\text{or}\qquad |x-y|.
\]

Hence  

\[
S' = S-x-y+(x+y)=S\quad\text{or}\quad
S' = S-x-y+|x-y| = S-2\min\{x,y\}.
\]

So the sum never increases; it stays the same if B writes the sum and it
decreases by an even amount if B writes the difference.

* The game stops as soon as  

\[
\max\{a_i\}> \frac{S}{2}\qquad\text{(condition (1))} 
\]

or when all numbers are zero (condition (2)).  
If the game stops after \(k\) rounds, the number of cookies that A receives is  

\[
N = N_0-k .
\]

Thus **A wants the game to finish as early as possible (few rounds),  
while B wants it to last as long as possible (many rounds).**

--------------------------------------------------------------------

### 2.  B’s optimal reaction in a round  

Assume the current multiset is \(\mathcal M\) and A has chosen the pair
\((x,y)\).

*If B writes the sum* \(x+y\) the total sum stays the same.
The only danger for B is that the new number becomes larger than half of the
new total sum, i.e. that  

\[
x+y>\frac{S}{2}.
\]

*If B writes the difference* \(|x-y|\) the total sum is reduced,
which never creates a new number larger than the old maximum, but it may
create many zeros and therefore may bring the game to condition (2)
prematurely.

Consequently **B’s optimal policy is**

* write the sum **unless** it would make the new number exceed \(S/2\);
* otherwise write the difference.

With this policy B always keeps the total sum equal to the initial sum
\(S_0=2020\) as long as it is safe to do so, and only reduces the sum when
forced to avoid condition (1).

For a board that contains a multiset `M` of non‑negative integers we define  

* `S(M) = sum(M)` – the total sum,  
* `m(M) = max(M)` – the largest element,  
* `|M|` – the number of entries.

The game stops as soon as  

* `m(M) > S(M)/2` (the “dominant” condition), or  
* `S(M)=0` (all entries are zero).

In every round the number of entries is reduced by one, therefore after `k` rounds the board contains `|M₀|-k` numbers, where `|M₀|` is the initial size (2020).  
Player A wants this final size to be as large as possible, player B wants it as small as possible.

### 1.   Invariants

* **The sum never increases.**  
  If B writes `x+y` the sum stays the same; if he writes `|x‑y|` the sum is reduced by `2·min(x,y)`.  

* **The condition “dominant element” depends only on the ratio `m(M)/S(M)`.**  
  When the sum is lowered while the maximum stays unchanged, the ratio grows, making the dominant condition easier to reach.  
  When the sum stays unchanged but the maximum grows, the ratio also grows.

Consequently, **B’s optimal move is always to write the sum `x+y`** – it never hurts him and often helps him create a larger maximum while keeping the total sum as large as possible (so that the ratio `m/S` can increase faster).  
Writing the difference can only *decrease* the sum without increasing the maximum, which makes the dominant condition *harder* to achieve.  
Hence in optimal play B will *always* choose `x+y`.
