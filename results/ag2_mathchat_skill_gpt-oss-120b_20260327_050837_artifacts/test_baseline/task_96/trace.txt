### 1.  What has to be proved  

For a given \(n\ge 2\) let  

\[
\pi :\{1,\dots ,n\}\longrightarrow \{1,\dots ,n\}
\]

be a permutation; the rook in row \(i\) is placed in column \(\pi(i)\).
A **\(k\times k\) empty square** is a set of consecutive rows  

\[
R=\{r,r+1,\dots ,r+k-1\}
\]

and consecutive columns  

\[
C=\{c,c+1,\dots ,c+k-1\}
\]

such that no pair \((i,\pi(i))\) lies in \(R\times C\).

We have to find the greatest integer \(k\) that is **always** present,
no matter which permutation \(\pi\) we start with.

--------------------------------------------------------------------

### 2.  Upper bound – a configuration that does not contain a larger empty square  

Take the *diagonal* configuration  

\[
\pi(i)=i\qquad (1\le i\le n).
\]

All rooks lie on the main diagonal.  
If we choose any set of \(k\) consecutive rows, the rooks that belong to
those rows occupy exactly the same \(k\) columns.
Consequently every set of \(k\) consecutive columns meets at least one
of those rooks, so a \(k\times k\) empty square cannot exist when  

\[
k>\frac{n}{2}.
\]

Indeed, if \(k\ge \bigl\lceil\frac{n+1}{2}\bigr\rceil\) then the
\(k\) columns occupied by the rooks already cover the whole board,
leaving no room for a disjoint block of \(k\) columns.
Hence no configuration can guarantee an empty square larger than  

\[
\left\lfloor\frac{n}{2}\right\rfloor .
\]

--------------------------------------------------------------------

We have to find the largest integer `k` such that **every** permutation matrix contains an empty `k×k` *contiguous* sub‑square (i.e. a set of consecutive rows and consecutive columns).

The problem is equivalent to:

> For a permutation `p` of `{1,…,n}` define

> Find `k = min_{p∈S_n} f(p)`.

\[

1. **Upper bound** – we exhibit a permutation for which no empty square larger than `⌊n/2⌋` exists.
