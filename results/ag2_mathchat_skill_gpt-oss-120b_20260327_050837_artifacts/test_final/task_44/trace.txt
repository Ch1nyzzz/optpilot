**Problem**

A rectangular garden has a length that is twice its width. The garden is surrounded by a uniform walkway of constant width \(w\) meters. The total area of the garden **plus** the walkway is 360 m², and the area of the garden alone is 180 m².

1. Find the dimensions (length and width) of the garden.

Output: We will write Python code using sympy to solve for w.```python
# Import sympy for symbolic calculations
import sympy as sp

# Define symbols
w = sp.symbols('w', real=True, positive=True)

# Garden dimensions: width W and length L = 2W
# Garden area = 180 => 2*W**2 = 180 => W = sqrt(90)
W = sp.sqrt(90)          # width of the garden
L = 2 * W                # length of the garden

# Total area including walkway is 360:
# (L + 2w)*(W + 2w) = 360
eq = sp.Eq((L + 2*w)*(W + 2*w), 360)

# Solve for w
solution_w = sp.solve(eq, w)
# Keep only the positive real solution
w_val = [sol for sol in solution_w if sol.is_real and sol > 0][0]

# Prepare the final results
garden_width = sp.simplify(W)          # width of garden
garden_length = sp.simplify(L)         # length of garden
walkway_width = sp.simplify(w_val)     # width of walkway

_result_ = {
    "garden_width": garden_width,
    "garden_length": garden_length,
    "walkway_width": walkway_width
}
print(_result_)
```
