Output: **Solution Overview**

We have  

* 25 white chips (indistinguishable) and 25 black chips (indistinguishable);  
* a \(5\times5\) board whose cells may contain at most one chip;  

and we must place chips so that  

1. every chip in a given row has the same colour, and every chip in a given column has the same colour;  
2. the placement is *maximal*: adding any further chip would break condition 1.

The task is to count all possible maximal placements.



--------------------------------------------------------------------

### 1.  Translating the conditions

Let  

* \(R_i\;(i=1,\dots ,5)\) be the colour of row \(i\) (if the row contains at least one chip);  
* \(C_j\;(j=1,\dots ,5)\) be the colour of column \(j\) (if the column contains at least one chip).

If a cell \((i,j)\) contains a chip, then the colour of that chip must equal both the colour of its row and the colour of its column, i.e.

\[
\text{chip at }(i,j) \Longrightarrow R_i = C_j .
\]

Consequences:

* If a row \(i\) contains a chip, then **every** column that also contains a chip must have the same colour as that row; otherwise the intersecting cell would be empty while the row and column share a colour, and we could place a chip there – contradicting maximality.
* Symmetrically, if a column \(j\) contains a chip, then every row that also contains a chip must have the same colour as that column.

Hence, as soon as at least one row (or column) is used, **all** rows and **all** columns must be used; otherwise an empty intersection would allow another chip.

Therefore every maximal placement uses **all five rows and all five columns**.



--------------------------------------------------------------------

### 2.  What does a maximal placement look like?

We now have a full set of rows \(R_1,\dots ,R_5\) and columns \(C_1,\dots ,C_5\), each coloured either **white (W)** or **black (B)**.

*If* \(R_i = C_j\) then the cell \((i,j)\) **must** contain a chip (otherwise we could add one of that colour).

# Count the number of maximal chip placements on a 5x5 grid
# according to the conditions described in the problem.

import itertools

def is_valid(row_colors, col_colors):
    """
    row_colors, col_colors are tuples of length 5 with entries 0 (white) or 1 (black).
    The placement is forced: a cell (i,j) is occupied iff row_colors[i] == col_colors[j].
    The configuration is maximal iff every row and every column has at least one occupied cell.
    """
    # For each row, there must be at least one column with the same colour
    for rc in row_colors:
        if not any(rc == cc for cc in col_colors):
            return False
    # For each column, there must be at least one row with the same colour
    for cc in col_colors:
        if not any(cc == rc for rc in row_colors):
            return False
    return True

# enumerate all colour assignments (0 = white, 1 = black)
all_rows = list(itertools.product([0,1], repeat=5))
all_cols = list(itertools.product([0,1], repeat=5))

count = 0
for r in all_rows:
    for c in all_cols:
        if is_valid(r, c):
            count += 1

print(count)   # expected result
```

```
