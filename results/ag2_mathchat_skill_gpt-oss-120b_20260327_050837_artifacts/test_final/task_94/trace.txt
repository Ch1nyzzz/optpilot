$$

Greedy George, on the other hand, chooses $x_{1}$ such that $\left|x_{1}\right|$ is as small as possible; among the remaining numbers, he chooses $x_{2}$ such that $\left|x_{1}+x_{2}\right|$ is as small as possible, and so on. Find the least possible constant $c$ such that for every positive integer $n$, for every collection of $n$ real numbers, and for every possible sequence that George might obtain, the resulting values satisfy the inequality $G \leqslant c D$.

### 1.  Upper bound – the greedy price is never more than twice the optimal price  

Denote the greedy partial sums by \(S_k=b_{1}+ \dots + b_{k}\) (\(S_0=0\)) and let  

\[
\Sigma =\sum_{i=1}^{n}a_i =\sum_{i=1}^{n}b_i .
\]

Because the final sum is the same for every ordering, the optimal price satisfies  

\[
D(A)\ge \frac{|\Sigma|}{2}\qquad\text{(otherwise the optimal ordering would have to cross the value }\pm\frac{|\Sigma|}{2}\text{).}
\tag{1}
\]

Let \(R_k\) be the set of numbers still unused after the first \(k-1\) greedy choices; its total is  

\[
\Sigma_k = \Sigma - S_{k-1}.
\]

George chooses \(b_k\in R_k\) that minimises \(|S_{k-1}+b_k|\).  
Among the numbers of \(R_k\) there is at least one whose sign coincides with the sign of \(\Sigma_k\); adding that number moves the current sum at most half‑way toward \(\Sigma_k\). Consequently  

\[
|S_k|
   =\min_{x\in R_k}|S_{k-1}+x|
   \le \frac12\bigl(|\Sigma_k|+|S_{k-1}|\bigr)
   =\frac12\bigl(|\Sigma|+|S_{k-1}|\bigr).          \tag{2}
\]

Iterating (2) from \(k=1\) up to any index and using \(|S_0|=0\) gives  

\[
|S_k|\le D(A)+\frac{|\Sigma|}{2}\qquad\text{for every }k.
\tag{3}
\]

Now (1) yields \(\frac{|\Sigma|}{2}\le D(A)\); inserting this into (3) we obtain  

\[
|S_k|\le D(A)+D(A)=2D(A)\quad\forall k,
\]

hence  

\[
G(A)=\max_{k}|S_k|\le 2D(A).
\]

Thus every greedy sequence satisfies \(G\le 2D\); consequently the required constant obeys \(c\le 2\).

import random
import math
from collections import Counter

def price(seq):
    """price of a sequence = max absolute partial sum"""
    s = 0
    best = 0
    for x in seq:
        s += x
        best = max(best, abs(s))
    return best

def greedy_price(multiset):
    """
    Returns the price G of ONE possible greedy ordering.
    If several numbers give the same minimal absolute sum,
    we simply pick the first one (any tie‑breaking works).
    """
    remaining = list(multiset)
    order = []
    cur = 0
    while remaining:
        # choose element that minimises |cur + x|
        best_val = None
        best_x = None
        for x in remaining:
            val = abs(cur + x)
            if best_val is None or val < best_val:
                best_val = val
                best_x = x
        order.append(best_x)
        cur += best_x
        remaining.remove(best_x)   # remove one occurrence
    return price(order)
