* \(l\) = length of the garden (meters)  

According to the statement

* The perimeter of a rectangle is \(2(l+w)\). We are told the fence length (the perimeter) is 54 m, so

We now have a system of two linear equations in the two unknowns \(w\) and \(l\). Solving it yields the dimensions, and the area follows from \(A = l\cdot w\).

```python
# Import the symbolic library
import sympy as sp

# Define the symbols for width (w) and length (l)
w, l = sp.symbols('w l', real=True)

eq1 = sp.Eq(l, w + 3)

solution = sp.solve((eq1, eq2), (w, l))
width_val = solution[w]

# Compute the area
```
