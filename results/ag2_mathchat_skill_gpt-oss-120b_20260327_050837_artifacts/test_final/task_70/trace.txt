import random
import itertools
import math

def min_diff_for_tuple(vals):
    """
    Given a list of 5 positive numbers, compute the smallest possible value of
        |a_i/a_j - a_k/a_l|
    where i,j,k,l are distinct indices.
    """
    n = len(vals)
    best = math.inf
    # all ordered pairs (i,j) with i!=j
    pairs = [(i, j) for i in range(n) for j in range(n) if i != j]
    # choose two pairs with disjoint indices
    for (i, j), (k, l) in itertools.product(pairs, repeat=2):
        if len({i, j, k, l}) == 4:          # all four indices distinct
            diff = abs(vals[i] / vals[j] - vals[k] / vals[l])
            if diff < best:
                best = diff
    return best

def search_random(trials=200000):
    """
    Random search to estimate the worst‑case (largest) minimal difference.
    Numbers are drawn from a wide log‑uniform distribution to explore
    extreme ratios.
    """
    worst = 0.0
    worst_tuple = None
    for _ in range(trials):
        # generate 5 positive numbers from 10^{U(-6,6)} (log‑uniform)
        vals = [10**random.uniform(-6, 6) for _ in range(5)]
        cur = min_diff_for_tuple(vals)
        if cur > worst:
            worst = cur
            worst_tuple = vals[:]
    return worst, worst_tuple

The garden is a rectangle \(30\text{ m} \times 20\text{ m}\).  
A uniform path of width \(x\) is built *inside* the perimeter, so the usable planting area becomes a smaller rectangle whose sides are reduced by \(2x\):

\[
\text{inner length}=30-2x,\qquad
\text{inner width}=20-2x.
\]

The inner rectangle’s area is given as \(400\text{ m}^2\):

\[
(30-2x)(20-2x)=400.
\]

Expanding and simplifying yields a quadratic equation in \(x\). Solving it gives two mathematical roots; only the one that fits the garden’s geometry (i.e., \(x\) must be less than half the smaller side, \(10\) m) is acceptable.

Below is a short Python script that uses **SymPy** to solve the equation and prints the valid width of the path.  

```python
# Import SymPy for symbolic mathematics
import sympy as sp

# Define the unknown width x (real, positive)
x = sp.symbols('x', real=True)

# Equation: (30 - 2x)*(20 - 2x) = 400
eq = sp.Eq((30 - 2*x)*(20 - 2*x), 400)

# Solve the quadratic equation
solutions = sp.solve(eq, x)

# Filter out the physically meaningful solution:
#   - width must be positive
#   - must be less than half of the garden's smaller side (10 m)
valid_solution = [sol.evalf() for sol in solutions if sol > 0 and sol < 10][0]

# Print the result
print(valid_solution)
```
