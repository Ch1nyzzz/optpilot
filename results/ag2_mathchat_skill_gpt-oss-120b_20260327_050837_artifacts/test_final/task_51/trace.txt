Input: **Problem Statement**

A rectangular garden has a length that is twice its width. The garden is to be surrounded by a fence, and a path of uniform width is to be built inside the fence, running along all four sides of the garden. The total area of the garden plus the path is 180 m², and the area of the garden alone is 120 m².

Determine the width of the path (in meters).
Output: **Solution Explanation**

Let  

* \(w\) = width of the garden (m)  
* \(l\) = length of the garden (m)  

The garden is a rectangle with  

\[
l = 2w ,\qquad \text{area}=l\cdot w = 2w^{2}=120\; \text{m}^{2}
\]

Hence  

\[
w^{2}=60\;\Longrightarrow\; w=\sqrt{60}\;(>0)
\]

A uniform path of width \(x\) (m) runs around the garden inside the fence.  
The outer rectangle (garden + path) therefore has dimensions  

\[
\text{outer width}=w+2x,\qquad 
\text{outer length}=l+2x = 2w+2x .
\]

Its total area is given as 180 m²:

\[
(w+2x)(2w+2x)=180 .
\]

Substituting \(w=\sqrt{60}\) and solving for \(x\) yields the required path width.

Below is a short Python script (using **sympy**) that carries out this calculation and prints the numeric value of \(x\) (rounded to three decimal places).  

```python
# Import sympy for symbolic algebra
import sympy as sp

# Define symbols
x = sp.symbols('x', real=True, positive=True)

# Garden dimensions
w = sp.sqrt(60)          # width of garden
l = 2 * w                # length of garden (twice the width)

# Equation for total area (garden + path)
eq = sp.Eq((w + 2*x) * (l + 2*x), 180)

# Solve the equation for x
solutions = sp.solve(eq, x)

# Keep the positive real solution
x_solution = [sol.evalf() for sol in solutions if sol.is_real and sol > 0][0]

# Round to three decimal places for a tidy answer
_result_ = round(float(x_solution), 3)

print(_result_)
```


Let  

* \(w\) = width of the whole garden (meters)  
* \(l = 2w\) = length of the whole garden (meters) – the length is twice the width.  
* \(x\) = uniform width of the inner path (meters).

The garden (including the path) is a rectangle of area  

\[
A_{\text{total}} = l\cdot w = 2w^2 = 180\;\text{m}^2 .
\]

Hence  

\[
w^2 = 90 \quad\Longrightarrow\quad w = \sqrt{90}=3\sqrt{10}\; \text{m}.
\]

The inner planting rectangle is obtained by removing the path from **both** sides of the outer rectangle, so its dimensions are  

\[
\text{inner width}= w-2x,\qquad 
\text{inner length}= l-2x = 2w-2x .
\]

Its area is given as  

\[
A_{\text{inner}} = (w-2x)(2w-2x)=80\;\text{m}^2 .
\]

Substituting the known value of \(w\) and solving for \(x\) yields the width of the path.

Below is a short Python script (using **SymPy**) that performs these calculations and prints the numerical value of \(x\) (rounded to three decimal places).  

```python
# -------------------------------------------------
# Solve for the width of the uniform path in the garden
# -------------------------------------------------

import sympy as sp

# Symbol for the unknown path width
x = sp.symbols('x', real=True)

# Known total area and inner area
total_area = 180          # m^2
inner_area = 80           # m^2

# From total area: width w satisfies 2*w**2 = total_area
w = sp.sqrt(total_area / 2)   # w = sqrt(90)

# Length is twice the width
l = 2 * w

# Equation for the inner rectangle area:
# (w - 2*x) * (l - 2*x) = inner_area
eq = sp.Eq((w - 2*x) * (l - 2*x), inner_area)

# Solve the quadratic equation for x
solutions = sp.solve(eq, x)

# Keep only the positive, physically meaningful solution (x must be < w/2)
x_val = [sol.evalf() for sol in solutions if sol.is_real and sol > 0][0]

# Round to three decimal places for a tidy answer
x_rounded = sp.N(x_val, 3)

# Print the result
print(x_rounded)
```
