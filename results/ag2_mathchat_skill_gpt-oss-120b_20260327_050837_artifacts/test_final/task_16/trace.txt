---

### 1.  Rotation formulas  

For \(\theta =60^{\circ}\),

\[
\cos\theta =\frac12 ,\qquad \sin\theta =\frac{\sqrt3}{2}.
\]

Clockwise rotation (\(-\theta\)) gives  

\[
\begin{aligned}
x' &= x\cos\theta + y\sin\theta =\frac12 x+\frac{\sqrt3}{2}y,\\[2mm]
y' &=-x\sin\theta + y\cos\theta =-\frac{\sqrt3}{2}x+\frac12 y .
\end{aligned}
\]

The point \((x',y')\) must satisfy the parabola equation:

\[
y' = (x')^{2}-4 .
\]

Together with the original equation \(y=x^{2}-4\) we have the system  

\[
\begin{cases}
y = x^{2}-4,\\[2mm]
-\dfrac{\sqrt3}{2}x+\dfrac12 y =\Bigl(\dfrac12 x+\dfrac{\sqrt3}{2}y\Bigr)^{2}-4 .
\end{cases}
\]

---

### 2.  Eliminate \(x\)

From the first equation \(x=\sqrt{y+4}\) (the fourth quadrant requires \(x>0\)).  
Insert this into the second equation, multiply by \(2\) and simplify:

\[
-\,\sqrt3\sqrt{y+4}+y = \frac12 (y+4)+\sqrt3\sqrt{y+4}\,y+\frac32 y^{2}-8 .
\]

After rearranging,

\[
3y^{2}-y-12 = -2\sqrt3\,(y+1)\sqrt{y+4}. \tag{1}
\]

---

### 3.  Remove the radical  

Square (1):

\[
\bigl(3y^{2}-y-12\bigr)^{2}=12\,(y+4)(y+1)^{2}.
\]

Expanding and simplifying yields a quartic polynomial:

\[
9y^{4}-18y^{3}-143y^{2}-84y+96=0.
\]

This factors nicely:

\[
(9y^{2}+9y-8)(y^{2}-3y-12)=0.
\]

Hence the possible \(y\)-values are  

\[
y=\frac{-3\pm\sqrt{41}}{6},\qquad
y=\frac{3\pm\sqrt{57}}{2}.
\]

Output: We will write Python code using sympy to solve.```python
# Import sympy for symbolic calculations
import sympy as sp

# Define symbols
w = sp.symbols('w', real=True)

# Given garden dimensions: length = 2*width, area = 180
# Let width = W, then 2*W*W = 180  ->  W^2 = 90
W = sp.sqrt(90)          # positive width of the garden
L = 2 * W                # length of the garden

# Walkway adds a uniform width w around the garden.
# Total outer rectangle dimensions: (L + 2w) by (W + 2w)
# Total area is 360 m².
eq = (L + 2*w)*(W + 2*w) - 360   # equation = 0

# Simplify the equation using the known value of L and W
eq_simplified = sp.simplify(eq)
eq_simplified
# Solve for w (positive root only)
solutions = sp.solve(eq_simplified, w)
solutions
# Filter the positive real solution
w_solution = [sol for sol in solutions if sol.is_real and sol > 0][0]

# Prepare the final results
_result_ = {
    "garden_width_m": sp.N(W, 6),          # width of garden (meters)
    "garden_length_m": sp.N(L, 6),         # length of garden (meters)
    "walkway_width_m": sp.N(w_solution, 6) # width of walkway (meters)
}
print(_result_)
