We have four “things” that a resident may own  

* D = diamond ring  
* G = golf clubs  
* S = garden spade  
* C = candy‑heart bag  

Every one of the 900 residents owns **C**.  
The other three items (D, G, S) are owned by some of the residents.

Let  

* \(a\) = residents who own **exactly one** of the four things (they own only C)  
* \(b\) = residents who own **exactly two** of the four things (C + one of D,G,S) – given \(b=437\)  
* \(c\) = residents who own **exactly three** of the four things (C + two of D,G,S) – given \(c=234\)  
* \(x\) = residents who own **all four** things (C + D + G + S) – this is what we need  

Because every resident owns C, the categories above exhaust all 900 residents:

\[
a+b+c+x = 900 \qquad\Longrightarrow\qquad a+x = 900-437-234 = 229\tag{1}
\]

### Breaking down the “exactly‑two’’ and “exactly‑three’’ groups  

**Exactly‑two group (size \(b=437\))**  
Each person in this group has C and *exactly one* of the other three items.  
Let  

* \(d_1\) = number with C + D  
* \(g_1\) = number with C + G  
* \(s_1\) = number with C + S  

Then  

\[
d_1+g_1+s_1 = 437 \tag{2}
\]

**Exactly‑three group (size \(c=234\))**  
Each person has C and *exactly two* of the other items.  
Let  

* \(dg\) = number with C + D + G

\[
dg+ds+gs = 234 \tag{3}
\]
