For a \(2n\times 2n\) board we look for the smallest number \(k\) of cells that can be *marked* so that  

* every domino in a tiling covers at most one marked cell, and  
* there is **exactly one** tiling satisfying this condition.

The problem is equivalent to asking for the smallest size of a *forcing set of monomers* for the square board.  
A monomer is a cell that is not allowed to be paired with another monomer; it must be paired with an ordinary (unmarked) cell.  

--------------------------------------------------------------------

### 1.  A lower bound

Each domino contains one black and one white square (the usual chessboard colouring).  
If we mark \(k\) cells, at most one of them can belong to the same domino, therefore the number of marked cells cannot exceed the number of dominoes:

\[
k\le \frac{(2n)^2}{2}=2n^{2}.
\tag{1}
\]

More importantly, a tiling is forced only when the marked cells “cut” all possible alternative cycles of dominoes.  
In a \(2n\times 2n\) board every cycle of dominoes contains at least \(2n\) cells of each colour (think of a horizontal “strip’’ of width 2 that can be flipped).  
Consequently at least one cell of each colour must be marked in each of the \(n\) horizontal strips of height 2, otherwise that strip could be tiled in two ways (all horizontal or all vertical).  

Hence we need at least one marked cell in each of the \(n\) strips, i.e.

\[
k\ge n .
\tag{2}
\]
